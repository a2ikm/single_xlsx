# SimpleXLSX

SimpleXLSX wraps [rubyXL](http://rubygems.org/gems/rubyXL) to treat single-sheet XLSX files like CSV.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'simple_xlsx'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simple_xlsx
